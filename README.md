# HackMatch AI 

HackMatch AI is an AI-powered platform that helps students discover hackathons that match their skills, interests, and experience level.

## Features
- AI-powered hackathon matching
- GCC and global hackathon discovery
- Team and partner finder
- Skill analysis and recommendations

## Tech Stack
- HTML
- CSS
- JavaScript

## Vision
HackMatch AI aims to make hackathon discovery easier for students and developers worldwide while supporting innovation aligned with Saudi Vision 2030.
